Wiki for DSharp.FrameWork.ContactGenerator

Library to generate Fake (Dummy) Data. Perfect for Unittesting, filling your objects or test database.

- Generates: Json, XML, SQL, CSV
- Country Specific Values: USA, Netherlands, Belgium.
- Uses real Cities, Zipcodes, First and Lastnames.

- Generator for creating lots of data.
- SimpleGenerator for filling single fields.

Current Available fields:
- BirthDay, City, Color, Country, Date, Email, FirstName, Gender, Guid, HexColor, HexNumber, Housenumber, Housenumberextension, Infix, IP4, IP6, ISBN, LastName, Lorum Ipsum, MAC, MobileNumber, Number, Password, PhoneNumber, PinCode Prefix, Row, Street, Suffix, Time, Username, Zipcode.

Note

Do not forget to change the properties on the resourcefile: Resources\ContactGenerator.dsrf
- Build Action: Content
- Copy to Output Directory: Copy if newer

It's also possible to move this file to a location you prefer.

Code Example

{
 Generator gen = new Generator();
 gen.Settings.Language = Language.US;
 gen.DownloadLatestPackage();

 gen.RecordDefinition.AddFirstName("MyFieldFirstName");
 gen.RecordDefinition.AddLastName("MyLastName");

 if (gen.Generate(20))
 {
 gen.ExportOptions.IgnoreEmpty = true;
 string json = gen.Export(@"D:\output.json");
 }
}

Do you want to generate from your own resources?
No Problem:

gen.Settings.UseLocalResources = true;
gen.Settings.PackageLocation = @"C:\PackageLocation";
